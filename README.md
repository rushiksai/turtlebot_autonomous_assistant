# TurtleBot Autonomous Assistant

## Overview

An autonomous service robot built using ROS 2 that navigates indoor environments, detects destinations, interacts with users, and performs tasks such as autonomous navigation and door interaction.

---

## Features

- Autonomous Navigation
- Goal Planning
- Door Detection
- Knock-Knock Interaction
- Return to Home
- Modular ROS2 Architecture
